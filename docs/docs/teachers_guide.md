# Module 1 교사 가이드

## 📚 개요
이 가이드는 경영학 학생들을 대상으로 한 3개 레벨의 게임화 실습을 진행하는 교사를 위한 것입니다.

### 학습 목표
- 수요 예측과 재고 관리의 중요성 이해
- 데이터 기반 의사결정 능력 배양
- 경영 시뮬레이션을 통한 실무 경험

## 🎮 각 레벨 개요

### Level 1: 도전, 김밥왕! (Newsvendor 문제)
**학습 목표**: 수요 불확실성 속에서의 최적 주문량 결정

**난이도**: 쉬움
**진행 시간**: 1-2시간
**핵심 개념**: 손실함수, 기대값 최소화

### Level 2: 편의점 점주의 가격 전략 (Price Elasticity)
**학습 목표**: 수요 탄성치를 이용한 가격 책정 전략

**난이도**: 중간
**진행 시간**: 2-3시간
**핵심 개념**: 수요 탄성치, 수익 최적화

### Level 3: 카페 점주의 마크다운 전략 (Dynamic Pricing)
**학습 목표**: 시간대별 가격 조정을 통한 폐기 손실 최소화

**난이도**: 어려움
**진행 시간**: 2-3시간
**핵심 개념**: 동적 가격 책정, 수익 극대화
