# 📚 Newsvendor Game - Gamified Business Simulation Platform

경영학 학생들을 위한 3단계 게임화 실습 플랫폼입니다.

## 🎮 플레이하기

[**플랫폼 시작하기**](./index.html) ← 여기를 클릭해서 게임을 시작하세요!

---

## 📖 모듈 구성

### Module 1: 수요 예측과 재고 관리

#### Level 1: 도전, 김밥왕! (Newsvendor 문제)
- **주제**: 불확실한 수요 속에서의 최적 주문량 결정
- **난이도**: 쉬움
- **소요시간**: 1-2시간
- **플레이**: [Level 1 게임 시작](./level1/index.html)

#### Level 2: 편의점 점주의 가격 전략 (Price Elasticity)
- **주제**: 수요 탄성치를 이용한 가격 책정
- **난이도**: 중간
- **소요시간**: 2-3시간
- **플레이**: [Level 2 게임 시작](./level2/index.html)

#### Level 3: 카페 점주의 마크다운 전략 (Dynamic Pricing)
- **주제**: 시간대별 가격 조정과 폐기 손실 최소화
- **난이도**: 어려움
- **소요시간**: 2-3시간
- **플레이**: [Level 3 게임 시작](./level3/index.html)

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript
- **Simulation**: 경영학 기반 수리 모델
- **Storage**: Firebase (학생 데이터, 리더보드)
- **Deployment**: GitHub Pages
